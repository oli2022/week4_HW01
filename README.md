# week4_HW01

## 第四週：期中考：打造全端 (FULL STACK) 網站架構

### 建立環境

```
$ npm install express-generator -g
```

### 建立專案

#### 可以先用 `express -h` 來查詢

```
$ express --no-view myApp
```

#### 進入 myApp

```
$ cd myApp

$ npm install
```

#### 新增需要的資料夾：

$\color{red}{connections}$ - 資料庫連結  
$\color{#FF3030}{controllers}$ - 控制器  
$\color{#FF3030}{models}$ - 模型  
$\color{#FF3030}{servers}$ - 訊息
